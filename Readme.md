strftime
========

strftime for JavaScript


Installation
============

npm install strftime


Usage
=====

    var strftime = require('strftime').strftime
    console.log(strftime('%D-%M-%Y %H:%M:%S'))

For details just see `man 3 strftime` as the format specifiers are identical.
